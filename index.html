🔗 Synchronisation GitHub

Chaque modification (créer/supprimer un tournoi, ajouter une ronde…) sera commitée automatiquement dans votre repo. Le JSON mis à jour est visible par tous les joueurs en quelques secondes.

Utiliser un token à portée public_repo (repo public) ou repo (repo privé). Stocké uniquement dans votre navigateur, jamais envoyé ailleurs qu'à l'API GitHub.

♟ Token Lichess (création des défis)

Un seul token suffit pour tout le club. Quand un joueur clique "Jouer cette partie", l'application crée automatiquement la partie sur Lichess et génère deux liens (un par joueur, avec la bonne couleur).

Permission requise : challenge:write. Stocké uniquement dans votre navigateur.
